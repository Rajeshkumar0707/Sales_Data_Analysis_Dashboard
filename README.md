📊 Sales Data Analysis

📌 Project Overview :
This project focuses on analyzing sales data to extract meaningful insights and support business decision-making. The analysis includes data cleaning, transformation, and visualization using Python, Excel, and Power BI.

🎯 Business Problem
Businesses need to understand:

. Which products generate the most revenue
. Sales trends over time
. Customer purchasing patterns
. Profitability of products

🛠 Tools & Technologies :
. Python (Pandas, NumPy)
. Excel
. Power BI
. MySQL (for database storage)

📂 Dataset
. Raw sales dataset (CSV format)
. Cleaned dataset after preprocessing (Sales_sheet)

🔍 Steps Performed
1. Data Cleaning
 . Removed missing values
 . Handled duplicates
 . Converted date columns
 . Fixed data types
2. Feature Engineering
 . Created Profit column (Sales - Cost)
 . Extracted Year and Month from date
3. Exploratory Data Analysis (EDA)
 . Summary statistics
 . Identified trends and patterns
 . Checked data distribution
4. Data Visualization
 . Built interactive dashboards in Power BI
 . Visualized:
   . Sales trends
   . Profit analysis
   . Category-wise performance
   
## ❓ Business Questions Solved
- Which category generates the highest sales and profit?
- What are the monthly sales trends?
- Which products are most profitable?
- Which region performs best?

## 📈 Key Insights
- Some categories contribute most of the revenue
- Sales show clear monthly trends
- Few products drive major profit
- Certain regions perform better than others

📊 Dashboard
Power BI dashboard includes:
. Sales overview
. Profit analysis
. Category performance
. Time-based trends

🚀 How to Run :
1. Load dataset in Python
2. Perform data cleaning and analysis
3. Export cleaned data
4. Import into Power BI
5. Create dashboard

📌 Conclusion :
This project demonstrates how data analysis can help businesses make informed decisions by identifying trends, improving profitability, and understanding customer behavior.
